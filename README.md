# Snapmaker_ita_gui
🇮🇹 Patch non invasiva che aggiunge la lingua Italiana alla GUI di Snapmaker sostituendo il dizionario russo. Nessuna compilazione richiesta, si installa con un singolo comando SSH. 🇬🇧 Non-invasive patch adding Italian language to Snapmaker GUI by replacing the Russian dictionary. No compilation required, installs via a single SSH command.
